Element follows the cursor with a magnetic pull effect. When the cursor enters the activation zone, the element smoothly shifts toward it. On exit, it eases back to its original position.

Works on buttons, icons, cards — any element.

### Options

- **strength** — pull intensity from 0 to 1 (default: 0.3)
- **distance** — activation zone radius in pixels (default: 150)
- **duration** — return animation duration in seconds (default: 0.4)
- **easing** — CSS easing function (default: ease-out)
