3D tilt effect that follows the cursor on hover. The element rotates toward the mouse position with perspective depth, creating a realistic 3D card effect.

Optional glare layer adds a light reflection that moves with the cursor.

### Options

- **max** — maximum tilt angle in degrees (default: 15)
- **perspective** — 3D depth in pixels (default: 1000)
- **scale** — scale factor on hover (default: 1.05)
- **speed** — transition duration in ms (default: 300)
- **glare** — enable light reflection overlay (default: false)
- **glare-opacity** — glare intensity from 0 to 1 (default: 0.3)
