Cursor-following spotlight that reveals content through a dark overlay. When the cursor enters the element, everything dims except a circle around the cursor.

Works on any element — cards, images, text blocks. The overlay fades in on hover and follows the cursor in real time.

### Options

- **size** — spotlight radius in pixels (default: 200)
- **border** — soft edge width in pixels (default: 100)
- **strength** — overlay darkness from 0 to 1 (default: 0.85)
- **color** — overlay color as RGB values (default: 0, 0, 0)
