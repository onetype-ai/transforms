Getting Started

Add a single script tag to your page. Each transform activates with an HTML attribute - no frameworks, no build tools, no complexity.

Installation

Copy the CDN link into your page head. All transforms load on demand when the page renders.

Configuration

Every transform is configured through ot-prefixed attributes directly in your markup. Set speed, color, direction, and more without writing JavaScript.

Attributes

Use ot-speed, ot-color, ot-direction and other attributes to control behavior. Values are parsed automatically.

Styling

Transforms use CSS variables for all visual properties. Override them to match your design system.

Deployment

Works on any platform - Webflow, WordPress, Squarespace, or custom HTML. No server required.
