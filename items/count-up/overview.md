Animated number counter that counts up when scrolled into view. Powered by CountUp.js.

Supports prefix, suffix, decimals, custom separators, easing control, and repeat animation on scroll.

### Options

- **end** — target number (default: 100)
- **start** — starting number (default: 0)
- **duration** — animation duration in seconds (default: 2)
- **decimals** — number of decimal places (default: 0)
- **prefix** — text before the number, e.g. "$" (default: none)
- **suffix** — text after the number, e.g. "%" (default: none)
- **separator** — thousands separator (default: ",")
- **decimal** — decimal separator (default: ".")
- **easing** — enable easing animation (default: true)
- **once** — animate only once, or repeat on every scroll into view (default: true)
