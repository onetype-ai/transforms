Image comparison slider that reveals before and after images by dragging a divider. Supports horizontal and vertical directions, hover mode, and optional labels.

Uses a native range input for full keyboard and screen reader accessibility.

### Options

- **position** — initial slider position in percent (default: 50)
- **direction** — "horizontal" or "vertical" (default: "horizontal")
- **label-before** — label text for the first image (default: none)
- **label-after** — label text for the second image (default: none)
- **hover** — slide on mouse hover instead of drag (default: false)
