Infinite scrolling marquee for logos, text, testimonials, and any content. Pure CSS animation with no external dependencies — lightweight, smooth, and GPU-accelerated.

Place any elements inside the marquee container and they will scroll infinitely. Content is automatically duplicated to create a seamless loop. Works with images, text, cards, or any HTML.

### Options

- **speed** — animation duration in seconds, lower is faster (default: 30)
- **direction** — scroll direction: "left", "right", "up", "down" (default: "left")
- **gap** — space between items in pixels (default: 40)
- **pause** — pause animation on hover (default: true)
- **fade** — gradient fade on edges (default: true)
